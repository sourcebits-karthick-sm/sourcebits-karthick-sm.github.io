Office Hour Rules

Glen Ridge employes work 7am-3pm, 9am-5pm, 10am-6pm or telecommute. Office hours are determined according to the conditions listed below.


  • All sales employees must work 9am-5pm unless conducting a sales call at a client's office.

  • All sales employees conducting a sales call at a clients office telecommutes.

  • All accounting employees work 9am-5pm except the collection manager who works 7am-3pm.

  • Production employees work 10am-6pm.

  • Any employee not previously mentioned works 9am-5pm.

INFORMATION

Guidelines for Making Copies


Copies are made using the main copier, the desktop copier, the fax machine, or the network printer. Guidlines for selecting the appropriate copying device are listed below.


  • A single copy of a 1-5 page document is made using the fax machine.

  • Two to ten copies of a 1-5 page document are made using the network printer unless there is only a hard copy of the original document in which case use the desktop copier.

  • Eleven to twenty copies of a 1-5 page document are made using the desktop copier unless the copies must be sorted or stapled in which case use the main copier.

  • More than 20 copies of any size document are made using the main copier.

  • All copies of a document with more than 5 pages are made using the main copier.
